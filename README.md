# Bairro Melhor 📍

App mobile para denúncias de problemas de infraestrutura urbana (buracos, lixo acumulado,
falta de iluminação, problemas de esgoto/água), desenvolvido em Flutter.
Atividade Extensionista II — Tecnologia Aplicada à Inclusão Digital (UNINTER).

## 📲 Download do APK

[⬇️ Baixar (APK)](https://github.com/gabrielsousaborges/bairro_melhor/releases/latest/download/app-release.apk)

## 🚀 Funcionalidades 
- Tela de login (mock)
- Listagem de denúncias
- Cadastro de nova denúncia (tipo, descrição, foto, localização)
- Exclusão de denúncias (swipe com confirmação)
- Persistência local dos dados (shared_preferences)
- Perfil do usuário com histórico e estatísticas

## 🛠️ Como rodar localmente

\`\`\`bash
git clone https://github.com/gabrielsousaborges/bairro_melhor.git
